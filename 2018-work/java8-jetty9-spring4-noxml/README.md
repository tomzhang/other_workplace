# Runnable war with Java8, Jetty9 and Spring4

This project was created to illustrate my problem posted on: http://stackoverflow.com/questions/28049350/runnable-war-with-maven-jetty9-spring4-and-no-web-xml

Build with: **mvn clean compile war:exploded antrun:run war:war**

Run with: **java -jar target/myapp.war** or drop **myapp.war** into %TOMCAT_HOME%/webapps

Feel free to use any part in Your own project :)
