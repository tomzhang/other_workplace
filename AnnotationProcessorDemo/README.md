# AnnotationProcessor
Java annotation processor demo. 

## 1.Compile - 编译

* 方法一  
执行`mvn clean compile`命令编译项目，可在Client子模块下可查看编译日志compile.log  
* 方法二  
执行`mvn package`打包项目后，终端运行Client子模块下compile.bat或compile.sh脚本。
